# Ahrefs Mini Checker (Go)

Простой инструмент командной строки (CLI) на Go для быстрой проверки основных SEO-метрик домена с использованием Ahrefs API v3.

### Автор

*   **Имя:** Частный SEO специалист
*   **Сайт:** [private-seo.com/ru](https://private-seo.com/ru/)

---

### Возможности

-   Получение Domain Rating (DR)
-   Количество бэклинков
-   Количество ссылающихся доменов
-   Количество органических ключевых слов
-   Примерный органический трафик

### Установка

1.  Убедитесь, что у вас установлен **Go** (версия 1.18 или выше).
2.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/ahrefs-checker.git
    cd ahrefs-checker
    ```
3.  Выполните сборку проекта:
    ```bash
    go build ./cmd/ahrefs-checker
    ```

### Использование

Для использования инструмента вам необходим **API токен** от Ahrefs.

Выполните команду из корневой папки проекта, передав ваш токен и домен для проверки:

```bash
./ahrefs-checker -token="ВАШ_API_ТОКЕН" -domain="example.com"
```

**Пример вывода:**

```
Получение данных для домена: example.com...

--- Результаты Ahrefs ---
Домен:           example.com
Domain Rating (DR): 92
Бэклинки:        1384591
Ссылающиеся домены: 74211
Органические ключевые слова: 412
Органический трафик: 1045
-------------------------
```

### Отказ от ответственности

Этот инструмент предоставляется "как есть". Автор не несет ответственности за использование API-юнитов вашего аккаунта Ahrefs. Каждый запрос к API потребляет юниты согласно вашему тарифному плану.
